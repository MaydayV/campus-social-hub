## ❓ 常见问题速查

### 操作类问题

<details>
<summary><b>如何添加新平台？</b></summary>

查看：[管理后台使用指南 - 平台管理](ADMIN_GUIDE.md#平台管理)

简要步骤：
1. 打开 admin.html
2. 点击【平台管理】
3. 点击【➕ 添加平台】
4. 填写信息并保存
</details>

<details>
<summary><b>如何批量导入账号？</b></summary>

查看：[Excel 模板格式说明](EXCEL_TEMPLATE.md)

简要步骤：
1. 导出 Excel 模板
2. 在 Excel 中编辑
3. 导入 Excel 文件
4. 生成配置文件
</details>

<details>
<summary><b>Excel 导入失败怎么办？</b></summary>

查看：[Excel 模板格式说明 - 常见错误](EXCEL_TEMPLATE.md#常见错误)

常见原因：
- 工作表名称不正确
- 数据格式错误
- URL 格式不正确
</details>

<details>
<summary><b>如何修改已有账号？</b></summary>

查看：[快速开始指南 - 场景3](QUICK_START.md#场景-3我要修改某个账号的链接)

简要步骤：
1. 打开管理后台
2. 找到要修改的账号
3. 点击【编辑】
4. 修改信息并保存
5. 生成配置文件
</details>

## 📌 使用提示

### 💡 阅读建议
- 📖 首次使用建议先阅读[快速开始指南](QUICK_START.md)
- 🔖 可以将常用文档加入浏览器书签
- 🔍 使用 Ctrl+F 在文档中搜索关键词
- 📝 遇到问题先查看 FAQ 部分

### ⏰ 时间安排
- **5分钟**: 快速了解 → [快速开始指南](QUICK_START.md)
- **15分钟**: 基础操作 → [管理后台使用指南](ADMIN_GUIDE.md) 部分章节
- **30分钟**: 完整学习 → 阅读所有文档
- **1小时**: 深入实践 → 尝试所有功能

### 🎓 学习技巧
1. **边看边做** - 打开管理后台，跟着文档操作
2. **做笔记** - 记录常用操作和个性化配置
3. **多实践** - 熟能生巧
4. **看示例** - 文档中的示例都是实际可用的
