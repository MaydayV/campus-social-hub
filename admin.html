📋 使用说明

方式一：Excel 编辑

  1. 点击【导出 Excel】按钮，下载当前配置
  2. 使用 Excel 或 WPS 打开文件进行编辑
  3. 编辑完成后，点击【导入 Excel】上传
  4. 系统会自动验证并更新数据

方式二：直接编辑

  1. 在【平台管理】【分类管理】【账号管理】中直接添加/编辑
  2. 编辑完成后，点击【生成配置文件】
  3. 下载生成的 config.js 文件
  4. 替换项目中的 js/config.js 文件即可

Sheet 3 - 账号配置：

平台标识 分类标识 账号名称 头像URL 链接URL 简介
wechat xiaoji 校团委 https://... https://... 发布校园活动

⚠️ 注意事项

  • 标识只能包含字母、数字和下划线
  • 标识必须唯一，不能重复
  • URL 必须以 http:// 或 https:// 开头
  • 头像图片建议尺寸：200x200 像素
